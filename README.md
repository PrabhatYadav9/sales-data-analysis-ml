Sales Data Analysis and ML project using Python, Pandas, and Scikit-learn

📊 Sales Data Analysis & Machine Learning

This project focuses on analyzing sales data and building a machine learning model to predict sales.
🔹 Key Steps:

 Data cleaning and exploration
 Outlier detection using IQR method
 Outlier handling using capping
 Feature engineering
 Model building using Random Forest
 Model evaluation using MAE and R² score

 🔹 Results:

 Model error reduced significantly after handling outliers
 Best performance achieved using capped data

 🔹 Tools Used:

 Python
 Pandas
 Matplotlib
 Scikit-learn

 🔹 Conclusion:

Proper data preprocessing plays a crucial role in improving machine learning model performance.
